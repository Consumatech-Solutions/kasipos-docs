# Project Architecture

KasiPOS is built with a resilient, offline-first architecture designed to operate in low-connectivity environments while maintaining a high-performance retail experience.

## Database Schema

```typescript
import Dexie, { Table } from 'dexie';

export interface Sale {
  id?: number;
  total: number;
  timestamp: number;
  items: any[];
  synced: boolean;
}

export class KasiDatabase extends Dexie {
  sales!: Table<Sale>;

  constructor() {
    super('KasiPOS_DB');
    this.version(1).stores({
      sales: '++id, timestamp, synced'
    });
  }
}

export const db = new KasiDatabase();
```
