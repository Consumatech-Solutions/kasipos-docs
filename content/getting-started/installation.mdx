# Installation

This guide walks you through running the **KasiPOS documentation website** on your machine. You will clone the repository, install dependencies with npm, and start the Next.js development server powered by Nextra.

## Clone the repository

```bash
git clone https://github.com/kasipos/kasipos-docs.git
cd kasipos-docs
```

If you contribute via a fork, clone your fork URL instead.

## Install dependencies

From the project root:

```bash
npm install
```

This reads `package-lock.json` and installs the same dependency versions used by the project.

## Start the development server

```bash
npm run dev
```

When the server is ready, open **[http://localhost:3000](http://localhost:3000)** in your browser. The root URL redirects into the docs; you will typically land at **`/docs/home`**.

## Verify it works

1. Confirm the docs home page loads at [http://localhost:3000/docs/home](http://localhost:3000/docs/home).
2. Optional: run a production build to catch MDX or type errors early:

```bash
npm run build
```

3. Optional: run the linter before opening a pull request:

```bash
npm run lint
```

To serve a production build locally, run `npm run build` then `npm run start`.
